# Paper's Tweaks

This mod is a collection of various gameplay tweaks I've made to make mine and my friends' lives easier in Lethal Company, and to make some _troublesome_ mechanics fairer for us _- looking at you, Kidnapper Fox._

All of the tweaks can be disabled or configured either by editing the config file or using LethalConfig in-game, though yb default they are all enabled with numbers I've found work best for my group of friends.

## Tweaks

### Vain Shroud Infestations

When enabled, moons will have a random chance (5% by default) to spawn with a large cluster of Vain Shrouds, usually big enough for a Kidnapper Fox to make into a nest.

This completely replaces the vanillla spawning mechanics where Vain Shrouds stay between days and grow over time. Instead, moons will only have Vain Shrouds if they get randomly picked to be infested and they will be gone once the day ends.

### Kidnapper Fox

- Reduces the health of the Kidnapper Fox to 3 HP instead of the default 7 HP, making them easier to kill with a shovel and one-shot with a shotgun.

### Butler

- Reduces the health of Butlers to 4 HP in multiplayer instead of the default 8 HP.
- Lowers the spawn limit for Butlers to 2 instead of the default 7, so moons like Dine won't be overrun by them.
- Prevents Butler Bees from spawning when a Butler dies.

## Bug Reports

If you find any bugs caused by the mod, please report them on the [GitHub issues](https://github.com/PaperbarkHorse/LC-PapersTweaks/issues) page.

I won't be taking any feature requests for this mod unless they relate to tweaks that have already been added.
